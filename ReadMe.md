## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Python (Flask)

**AI Model:** Hugging Face Transformers (BlenderBot)






## How to Run

Clone the project

```bash
  git clone https://github.com/EesaraMegasooriya/MedicBot
```

Go to the project directory

```bash
  cd my-project
```

First Run finetune.py

```bash
  python3 finetune.py
```

Start the bot

```bash
  python3 bot.py
```
